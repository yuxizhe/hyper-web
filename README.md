# 前端项目
## 安装依赖
```bash
cd frontend
pnpm install
```

## 脚本描述

### 开发启动
```bash
npm run dev
```

### 打包

```bash
npm run build
```

# 后端服务

```
pip install "fastapi[standard]"
```

## 开发启动
```bash
fastapi dev main.py
```

Server started at http://127.0.0.1:8000

Documentation at http://127.0.0.1:8000/docs
